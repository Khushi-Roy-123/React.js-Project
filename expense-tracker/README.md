Keep track of your income and expenses with this React-based tracker.

## Features
- **Balance Tracking**: Real-time calculation of total balance.

- **Transaction History**: List of past transactions with delete functionality.
- **Context API**: Uses React Context for global state management.

## Tech Stack
- React
- Vite
- Context API

## Setup & Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the tracker:
   ```bash
   npm run dev
   ```
