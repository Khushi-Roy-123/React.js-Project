A gamified Todo List built with React and Vite. Turn your daily tasks into an RPG adventure!

## Features
- **Gamification**: Earn XP for adding and completing tasks.
- **Level Up System**: Level up as you complete more quests.
- **Quest Log**: Manage your tasks with a clean, dark-mode interface.
- **Confetti Celebration**: Visual rewards for leveling up.

## Tech Stack
- React
- Vite
- CSS3 (Custom Dark Theme)
- `react-confetti`
- `uuid`

## Setup & Run
1. Clone the repository (or navigate to folder).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the game:
   ```bash
   npm run dev
   ```
