# React.js Project Collection

This repository contains a collection of 5 distinct React.js applications, each demonstrating different concepts and UI designs.

## üöÄ Projects

### 1. [Expense Tracker](./expense-tracker)
A comprehensive expense tracking application to manage your finances.
- **Features**: Add income/expenses, view balance, transaction history.
- **Tech**: React Context API, Global State management.
- **UI**: Custom CSS with glassmorphism elements.

### 2. [Weather App](./weather-app)
A real-time weather widget powered by OpenWeatherMap API.
- **Features**: Search by city, dynamic backgrounds (Sunny/Rainy/Cold), live temperature & humidity data.
- **Tech**: React, Fetch API, Environment Variables.
- **UI**: Modern Glassmorphism design with responsive gradients.

### 3. [Lottery Game](./lottery-game)
A fun, interactive lottery simulation game.
- **Features**: Buy tickets, winning condition logic (Sum = 15), confetti celebration effects.
- **Tech**: React State hooks, component composition.
- **UI**: Playful design with animations.

### 4. [Todo Game](./todo-game)
(Description based on typical Todo App, assuming gamification elements or standard todo)
A task management tool to keep track of daily activities.
- **Features**: Add, delete, and mark tasks as complete.
- **Tech**: React State, Props.

### 5. [Calculator](./calculator)
A fully functional calculator application.
- **Features**: Standard arithmetic operations with a clean interface.
- **Tech**: React event handling.
- **UI**: Neumorphic/Clean design.

