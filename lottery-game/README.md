A simple yet fun Lottery Game built with React and TailwindCSS.

## Features
- **Interactive Ticket**: Generates random lottery numbers interactively.
- **Winning Condition**: Checks if the sum of digits equals the winning number (15).
- **Visual Feedback**: Confetti and animations when you win.
- **Responsive UI**: Built with TailwindCSS for a modern look.

## Tech Stack
- React
- Vite
- TailwindCSS
- `react-confetti`

## Setup & Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the game:
   ```bash
   npm run dev
   ```
